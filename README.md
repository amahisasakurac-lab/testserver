## 3. Xdomain 無料サーバー メモ

> ※ 契約時に最新の公式情報を確認すること。以下は一般的な傾向。

- Xdomain の**無料レンタルサーバー**を利用予定
- 無料版は主に **FTP でのファイル転送**（SSH は基本的に非対応）
- 用途に応じてプランが分かれる
  - HTML / CSS / JS 中心の静的サイト向け
  - PHP / MySQL が使えるプログラム向け
- 提供されるサブドメイン、または独自ドメインで公開可能

## 6. 自動デプロイ（将来メモ・今はやらない）

Xdomain 無料サーバーは SSH 非対応の想定なので、将来自動化するなら **FTP 方式**が現実的。

- GitHub Actions の FTP デプロイ（例: FTP-Deploy-Action 系）を使う
- FTP 接続情報は **GitHub Secrets** に登録し、コードには書かない
- まずは手動 FTP アップロードで運用し、慣れてから自動化する

---

## 7. セキュリティ・運用メモ

- FTP パスワードや接続情報は **リポジトリに絶対に含めない**（`.gitignore` + Secrets）
- `.env` や認証情報ファイルは `.gitignore` で必ず除外
- GitHub リポジトリは原則 **Private**
- テスト環境は検索避け（`robots.txt` / Basic 認証 等）
